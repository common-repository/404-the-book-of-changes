*How does our plugin work*

Plugin «404 The Book of Changes» is a selection of 8 out of the 64 points in the random number generator. Every choice is unique, each field is unique, each result is unique. The result of a choice is a small piece of text from a very ancient treatise «The Book of Changes».

each choice is unique not by chance:

*field changes every time you refresh the page;
*each character has a random meaning and it’s hidden;
*only you can select the hidden meaning.

This approach allows you to get unique combination of 2 ^ 64 characters, which is equal to 18 446 744 073 709 551 616 points.

*Цель нашего плагина — Как бы реклама :-)*

Если пользователь совершил случайную ошибку и перешел на страницу, которой не существует, то чаще всего его встретит страница с грустным смайликом. Это интересно? Нет.

Если пользователь совершил вообще случайный переход, то грустный смайлик на 404 странице ему понравится? Нет.

Если пользователю станет скучно, то много ли способов его заинтересовать? Нет.

Мы предлагаем один ответ «Да» на все эти вопросы. Мы предлагаем бесплатный плагин — Книга перемен.

*Работа нашего плагина*

Плагин «404 Книга перемен» представляет собой выбор 8 из 64 значений в поле генератора случайных чисел. Каждый выбор — уникален, каждое поле — уникально, каждый результат — уникален. Результат выбора — это небольшой отрывок текста очень древнего трактата «Книга перемен».

Мы не зря говорим, что каждый выбор уникален:

*поле меняется каждый раз при обновлении страницы
*каждый символ задается случайным значением и скрыт
*только Вы можете выбрать скрытое значение

Подобный подход позволяет получать уникальное сочетание из 2^64 символов, что равно 18 446 744 073 709 551 616 значениям.
